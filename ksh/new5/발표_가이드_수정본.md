## 1. 주제 선정 이유

편의점 출점·운영 판단이 현장에서는 "사람 많아 보인다" 같은 경험/감각에 의존하는 경우가 많다.
그런데 실제 매출은 유동인구 외에도 점포 밀집도, 상권 성격 등 여러 요인이 동시에 작동하므로, 
**무엇이 더 중요한지 정량 근거로 설명할 필요가 있다.**

→ 본 분석은 데이터 기반으로 매출 결정 요인을 검증하고, 
   신규 출점 및 기존 점포 운영에 활용할 수 있는 인사이트를 도출한다.

## 4. 문제 정의 및 가설

### 연구 질문
> **"서울시 편의점 매출을 결정하는 핵심 요인은 무엇인가?"**

### 가설 구조

#### [메인 가설 - 4개]

**H1: 유동인구 → 매출**
- H0-1: 유동인구와 매출은 유의한 관계가 없다.
- Ha-1: 유동인구가 증가할수록 매출이 증가한다.
- 검증방법: 상관분석, 회귀분석

**H2: 점포수(밀집도) → 매출**
- H0-2: 점포수와 매출은 유의한 관계가 없다.
- Ha-2: 점포수가 증가할수록 매출이 증가한다.
- 검증방법: 상관분석, 회귀분석
- ※ 주의: 역인과 가능성 (매출↑ → 점포↑) 존재

**H3: 상권유형 → 매출 (회귀계수 유의성)**
- H0-3: 상권유형은 매출에 유의한 영향이 없다.
- Ha-3: 상권유형에 따라 매출이 유의하게 다르다.
- 검증방법: 회귀분석 (더미변수 계수 유의성)

**H4: 상권유형 간 매출 차이 (그룹 간 비교)**
- H0-4: 상권유형 간 평균 매출 차이가 없다.
- Ha-4: 최소 하나의 상권유형 간 매출 차이가 있다.
- 검증방법: ANOVA + Tukey HSD 사후검정

#### [사전 점검 - 다중공선성]

**유동인구 ↔ 점포수 관계**
- 목적: 독립변수 간 다중공선성 확인
- 기준: VIF < 5 → 문제없음, VIF 5~10 → 주의, VIF > 10 → 심각
- 결과: VIF < 5 확인 → 두 변수 모두 모델에 포함 가능

## 7. EDA (탐색적 데이터 분석)

### 7.1 목적
- 회귀분석 전 가설 방향성 사전 확인
- 다중공선성, 이상치 등 분석 리스크 점검
- **시계열 트렌드 파악**

### 7.2 주요 발견

**Y(매출) 분포**
- 평균: 27.1억원, 중앙값: 17.2억원
- 왜도: 3.92 (오른쪽 꼬리 긴 분포)
- → 고매출 지역이 소수 존재, 이분산 가능성

**분기별 매출 추이 (신규)**
- 분석 기간: 2022Q1 ~ 2025Q3 (15개 분기)
- 전체 기간 평균 매출 추이 확인
- 상권유형별 성장/하락 트렌드 비교
- → 시간에 따른 패턴 파악

**H1 예비검증 (유동인구→매출)**
- 상관계수: r = 0.499 (p < 0.001)
- 해석: 중간 강도의 양의 상관
- → H1 예비 지지

**H2 예비검증 (점포수→매출)**
- 상관계수: r = 0.807 (p < 0.001)
- 해석: 강한 양의 상관
- → H2 예비 지지 (단, 역인과 주의)

**H3/H4 예비검증 (상권유형→매출)**
- 상권유형별 평균 매출 차이 확인
- 관광특구(82.6억) > 발달상권(62.6억) > 전통시장(28.5억) > 골목상권(24.9억) > 미분류(13.3억)
- ANOVA F=206.5 (p < 0.001)
- → H3/H4 예비 지지

**다중공선성 사전점검**
- 유동인구 ↔ 점포수 상관: r = 0.485
- → 다중공선성 심각하지 않음 (VIF 분석으로 확정)

## 8. 분석 방법론

### 8.1 통계 분석
| 분석 | 목적 | 방법 |
|------|------|------|
| 상관분석 | H1, H2 예비검증 | Pearson 상관계수 |
| ANOVA | H4 그룹 간 차이 | F-검정 + Tukey HSD |
| 회귀분석 | H1, H2, H3 통합 검증 | OLS + Robust SE (HC3) |
| **표준화 계수** | **변수 간 영향력 비교** | **표준화 회귀분석** |
| VIF | 다중공선성 진단 | 기준: VIF < 5 |
| Breusch-Pagan | 이분산 검정 | 잔차 진단 |

### 8.2 표준화 회귀계수 해석
- **목적**: 서로 다른 단위의 변수들의 상대적 영향력 비교
- **방법**: 모든 변수를 평균=0, 표준편차=1로 표준화 후 회귀분석
- **해석**: |β|가 클수록 매출에 더 큰 영향
- **활용**: "유동인구 vs 점포수 중 뭐가 더 중요?" 질문에 답변 가능

### 8.2 머신러닝 예측
| 모델 | 목적 | 평가지표 |
|------|------|----------|
| Linear Regression | 베이스라인 | R², MAE |
| Random Forest | 예측 성능 | R², MAE, OOB |
| Gradient Boosting | 예측 성능 | R², MAE, CV |

### 8.3 시나리오 분석
- What-if: 유동인구/점포수/상권유형 변화 시 매출 예측
- 저평가/고평가 지역 식별: 잔차 분석
- 신규 출점 추천: 점포당 유동인구 + 예측 매출 기반

## 10. 핵심 인사이트

### 인사이트 1: 상권유형이 매출의 핵심 결정 요인
- 관광특구: 골목상권 대비 +27억원
- 발달상권: 골목상권 대비 +15억원
- 같은 조건이라면 상권유형이 매출을 좌우

### 인사이트 2: 저평가 지역 = 성장 기회
- 조건(유동인구, 점포수) 대비 매출이 낮은 지역 다수 존재
- 운영 개선 또는 신규 진입 기회

### 인사이트 3: 점포 효율성은 상권유형마다 다름
- 점포수가 많다고 점포당 매출이 높은 것은 아님
- 상권유형별 차별화 전략 필요
