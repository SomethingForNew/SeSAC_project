[4] 주요 발견사항
======================================================================

1. 상권유형별 매출 격차 매우 큼:
   발달상권(65.8억) : 골목상권(25.4억) = 2.6배
   발달상권(65.8억) : 미분류(13.3억) = 5.0배
   
   → 입지가 매출의 핵심 결정 요인
   → 5.7%에 불과한 발달상권이 고매출 담당

2. 매출 결정 요인 우선순위:
   1순위: 점포수 (상관계수 r=0.81)
   2순위: 상권유형
   3순위: 유동인구 (상관계수 r=0.49)
   
   → 점포수가 매출 변동의 65% 설명
   → 유동인구는 24%만 설명 = 보조 지표
   → 유동인구의 '양'보다 '질'(구매력)이 중요

3. 시계열 추세 - 2023년 정점 후 하락:
   2022: 27.5억 (+기준)
   2023: 27.9억 (+1.3%) ← 정점
   2024: 26.9억 (-3.5%)
   2025: 25.8억 (-4.1%)
   
   4년간 누적 변화:
   - 매출: -6.0%
   - 점포수: +8.9%
   - 유동인구: -3.1%
   
   → 점포는 늘었는데 매출은 줄어듦
   → 점포당 평균 매출 -13.6% 감소
   → 과당 경쟁 심화

4. 공간적 특성:
   - 14개 행정동은 점포 0개 (하지만 매출 기록 존재)
     이유: 분기 말 기준 0개, 분기 중 단기 영업
   - 역삼1동 51개로 최대 (평균의 8.3배)
   - 강남/여의도/명동 등 업무지구에 점포 집중

5. 유동인구 vs 매출의 비선형 관계:
   - 발달상권: 유동인구 적어도 고매출
     (539만명 but 평균 66억)
   - 골목상권: 유동인구 많아도 중간 매출
     (581만명 but 평균 25억)
   
   → 유동인구의 '질'(구매력, 소비 목적)이 핵심
   → 단순 통과 인구는 매출 기여 낮음

[6] 상관관계 요약

매출 vs 점포수:      r = 0.8065 (강한 양의 상관)
                   → 점포수가 매출의 가장 강력한 예측 변수
                   → R² = 0.65 (매출 변동의 65% 설명)

매출 vs 유동인구:    r = 0.4940 (중간 양의 상관)
                   → 유동인구는 보조 지표
                   → R² = 0.24 (매출 변동의 24%만 설명)

점포수 vs 유동인구:  r = 0.4785 (중간 양의 상관)
                   → 점포 입지가 단순 인구밀도만 따르지 않음

해석:
  - 점포수 = 상권 규모의 직접적 지표
  - 편의점이 많은 곳 = 수요가 검증된 곳
