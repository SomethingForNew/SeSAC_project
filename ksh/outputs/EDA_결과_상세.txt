  - 평균 점포수: 6.25개
  - 평균 유동인구: 5,504,944명
  - 전년 대비: 매출 -4.1%, 점포 -2.5%, 유동인구 -0.9%

해석:
  - 2022→2023: 매출 소폭 증가 (27.5→27.9억, +1.3%)
    코로나 회복기, 소비 심리 개선
    
  - 2023→2024: 매출 감소 시작 (27.9→26.9억, -3.3%)
    금리 인상, 경기 둔화 영향
    점포는 증가하는데 매출은 감소 = 경쟁 심화
    
  - 2024→2025: 매출 감소 지속 (26.9→25.8억, -4.1%)
    경기 침체 본격화
    점포 감소 시작 = 일부 폐업 발생
  
  - 4년간 누적: 매출 -6.0%, 점포 +8.9%, 유동인구 -3.1%
  
  시사점:
  1. 점포 증가(+8.9%)에도 매출 감소(-6.0%)
     → 점포당 평균 매출 -13.6% 감소
     → 과당 경쟁 심화
  
  2. 유동인구 감소(-3.1%) < 매출 감소(-6.0%)
     → 유동인구 감소가 전부가 아님
     → 1인당 지출액도 감소 = 소비 패턴 변화
     → 온라인 쇼핑, 배달 서비스 확대 영향
  
  3. 2023년이 정점
     → 향후 추가 하락 가능성
     → 보수적 예측 필요


[6] 상관관계 분석
======================================================================

상관계수 매트릭스:
                    당월_매출_금액  점포_수  총_유동인구_수
당월_매출_금액          1.0000  0.8065  0.4940
점포_수                0.8065  1.0000  0.4785
총_유동인구_수          0.4940  0.4785  1.0000

해석:

  1. 매출 vs 점포수: r = 0.8065 (강한 양의 상관)
     → 점포가 많을수록 매출이 높음
     → 결정계수 R² = 0.65 → 점포수가 매출 변동의 65% 설명
     → 가장 강력한 예측 변수
     
     의미:
     - 점포수 = 상권 규모의 직접적 지표
     - 편의점이 많은 곳 = 수요가 많은 곳
     - 점포 1개 증가 시 약 4~5억 매출 증가 예상
     
     주의:
     - 인과관계가 아님 (점포를 늘린다고 매출이 늘지 않음)
     - 수요가 많은 곳에 점포가 많이 입점한 것

  2. 매출 vs 유동인구: r = 0.4940 (중간 양의 상관)
     → 유동인구가 많다고 매출이 비례적으로 높지 않음
     → 결정계수 R² = 0.24 → 유동인구가 매출의 24%만 설명
     → 점포수보다 예측력이 낮음
     
     의미:
     - 유동인구의 '양'보다 '질'(구매력)이 중요
     - 발달상권: 유동인구 적어도 고매출 (고소득층)
     - 주거지역: 유동인구 많아도 저매출 (생활 통행)
     
     예외 케이스:
     - 통과 인구 (역, 버스정류장 등) → 구매 전환율 낮음
     - 특수 시간대 집중 (출퇴근) → 24시간 평균보다 효과 적음

  3. 점포수 vs 유동인구: r = 0.4785 (중간 양의 상관)
     → 점포 입지가 단순히 인구밀도만 따르지 않음
     
     의미:
     - 상권 특성에 따라 점포 밀집도가 다름
     - 업무지구: 낮 인구 많지만 밤 인구 적음 → 점포 중간 수준
     - 주거지역: 24시간 안정적 수요 → 점포 고른 분포
     - 발달상권: 높은 수익성 → 유동인구 대비 점포 과밀

종합:
  - 매출 예측 우선순위: 점포수 > 상권유형 > 유동인구
  - 3가지 변수를 함께 고려해야 정확한 예측
  - 단일 변수로는 한계 존재
