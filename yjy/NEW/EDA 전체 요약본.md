- **지역**: 서울시 425개 행정동
- **데이터**: 서울시 상권분석서비스 - 길단위인구

|------|------|

| 행정동 수 | 425개 |

### 유동인구 통계
- **평균**: 5,518,692명
- **최소**: 7,864명
- **최대**: 22,048,225명

### 시각화
- 분기별 유동인구 추이
- 시간대별 유동인구 (00~06시, 06~11시 등)
- 요일별 유동인구 (주중 vs 주말)

### 적합성 판단
```
✅ 데이터 양: 적합 (5,100개 > 1,000개)

```

### 활용 컬럼
- `총_유동인구_수`: 기본 독립변수
- 시간대별/요일별: 세부 분석 가능

→ **독립변수(X)**로 사용 가능!

## 🏪 2. 상권유형 (eda_district_type.ipynb)

### 분석 대상
- **지역**: 서울시 행정동
- **데이터**: 서울시 상권분석서비스 - 영역(상권)
- ⚠️ **기간 정보 없음** (스냅샷 데이터)

### 상권유형 4가지
| 유형 | 설명 |
|------|------|
| 골목상권 | 주거지 근처 소규모 상권 |
| 발달상권 | 대형 상업지역 (강남역, 홍대 등) |
| 전통시장 | 재래시장 |
| 관광특구 | 명동, 이태원 등 |

### 확인 내용
| 항목 | 결과 |
|------|------|
| 상권유형 | 4가지 |
| 결측치 | 0% |

### 시각화
- 상권유형별 파이차트
- 자치구별 상권유형 스택바 차트
- 상권유형별 영역 면적 박스플롯

### 적합성 판단
```
✅ 데이터 양: 적합 (500개 이상)
✅ 상권유형: 적합 (4가지 > 3가지)
✅ 행정동 수: 적합 (100개 이상)
✅ 결측치: 적합 (0%)
```

### 활용 방안
- 행정동별 상권유형 개수를 독립변수로 사용
- 또는 더미변수로 변환하여 회귀분석 가능

→ **독립변수(X)**로 사용 가능!
