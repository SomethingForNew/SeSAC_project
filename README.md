# 참고 가이드(수정)메인 질문:
"강남 3구(강남/서초/송파) 아파트 실거래가는 정말로 계속 비싸지고 있는가?"

서브 질문:
"어떤 요인들이 가격에 (+) 또는 (-) 영향을 미쳤는가?"

- 소득수준
- 금리
- 세금/규제
- 뉴스 감성

필요한 데이터

1. 아파트 실거래가(API)

- 대상: 강남구, 서초구, 송파구
- 기간: 최근 3년 (2023~2025)
- 출처: 서울 열린데이터 광장

1. 뉴스감성(크롤링)

- 대상: 중앙일보 부동산 뉴스
- 기간: 최근 3년(2023~2025)
- 목적: 뉴스 긍정/부정이 가격에 미친 영향

1. 경제 요인 데이터(API)

- 금리 데이터 → https://ecos.bok.or.kr
  - 검색 키워드: 기준금리, 주택담보대출금리
- 세금/규제 데이터 →
- 소득수준 데이터 → KOSIS, 서울 열린 데이터광장, 국세청 통계

분석1: 강남 3구, 정말 비싸지고 있나?

목적: 기본 현황 파악

분석 내용:

- 월별 평균 실거래가 추이 (2023~2025)
- 구별 비교 (강남 vs 서초 vs 송파)
- 규모별 추이 (소형/중형/대형)
- 증가율 계산 (%, 절대금액)

핵심 질문:

- 정말 오르고 있나? → YES/NO
- 얼마나 올랐나? → X% 또는 X억
- 어느 구가 제일 많이 올랐나?

시각화 (인터랙티브):

- 시계열 라인 차트 (구별 색깔)
- 증가율 바 차트
- 지도 히트맵

분석2: 소득이 높아지면 강남 3구 집값도 오르나?

목적: 소득이 집값에 미친 영향 파악

분석 내용:

- 소득 vs 실거래가 상관관계
- 소득 증가율 vs 집값 증가율
- PIR(소득대비집값) 변화 추이
- 시차 분석 (소득 변화 후 몇 개월?)

핵심 질문:

- 소득이 오르면 집값도 오르나? → 상관계수
- 소득보다 집값이 더 빠르게 올랐나? → 괴리도
- 소득 변화가 집값에 선행하나? → 시차

시각화 (인터랙티브):

- 이중축 차트 (소득+집값)
- 산점도 (correlation)
- PIR 추이

분석3: 금리가 오르면 실거래가가 떨어졌나?

목적: 금리가 집값에 미친 영향 파악

분석 내용:

- 금리 vs 실거래가 상관관계
- 금리 인상/인하 시점 전후 비교
- 시차 분석 (금리 변화 후 몇 개월?)
- 금리 구간별 가격 변화

핵심 질문:

- 금리가 오르면 집값이 떨어지나? → 음의 상관관계?
- 반응 속도는? → 1~3개월?
- 강남 3구도 금리에 민감한가?

시각화 (인터랙티브):

- 이중축 차트 (금리+집값)
- 금리구간별 가격변화
- 시차 상관분석 그래프

분석4: 뉴스 감성 영향

목적: 뉴스 긍정/부정이 집값에 미친 영향

분석 내용:

- 감성지수 vs 실거래가 상관관계
- 긍정/부정 뉴스 증가 시점 전후 비교
- 시차 분석
- 구별 민감도 차이

핵심 질문:

- 긍정 뉴스가 많으면 집값 오르나?
- 반응 속도는?
- 어느 구가 뉴스에 가장 민감한가?

시각화 (인터랙티브):

- 이중축 차트 (감성지수+집값)
- 긍정/부정 비교
- 시차 상관분석

분석5: 종합 영향력 분석(머신러닝)

목적: 소득/금리/뉴스 중 가장 큰 영향 요인 찾기

방법:

- 다중회귀분석 또는 랜덤포레스트
- Feature Importance 계산
- SHAP value 분석 (선택)

변수 구성:

- X (독립변수):
  - 소득수준
  - 금리
  - 감성지수
  - 시간(트렌드)
- y (종속변수):
  - 실거래가

핵심 질문:

- 가장 큰 영향 요인은? → 1위, 2위, 3위
- 각 요인의 기여도는? → %로 표현
- 예측 정확도는? → R² score

시각화 (인터랙티브):

- Feature Importance 바 차트
- 워터폴 차트 (각 요인 기여분)
- 예측 vs 실제 비교
- SHAP value plot

최종 인터랙티브 시각화 5개

1. 메인 대시보드 (분석1)

- 강남 3구 실거래가 추이
- 드롭다운으로 구 선택
- 슬라이더로 기간 조절

1. 소득-집값 상관분석 (분석2)

- 이중축 시계열 차트
- PIR 트렌드
- 애니메이션 산점도

1. 금리-집값 상관분석 (분석3)

- 이중축 시계열 차트
- 금리 구간별 가격 변화
- 시차 상관계수 그래프

1. 뉴스 감성-집값 분석 (분석4)

- 이중축 시계열 차트
- 긍정/부정 단어 빈도
- 구별 민감도 비교

1. 종합 영향력 대시보드 (분석 5)

- Feature Importance 바 차트
- 워터폴 차트
- 예측 vs 실제
- 요인별 슬라이더로 시뮬레이션
